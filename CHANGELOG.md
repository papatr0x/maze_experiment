# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-07-05

### Added
- Randomized Depth-First Search (DFS) maze generation.
- Visual traversal progress on screen.
- Highlighted current position during traversal.
- Restart functionality (press 'R').
- Timing control to slow down the animation.
- Basic project structure with `uv` support.
- Comprehensive `README.md` and MIT `LICENSE`.
- Enhanced `.gitignore` and `pyproject.toml` metadata.

### Fixed
- Fixed several deprecation and code style warnings related to Arcade 3.x.
- Corrected method signatures for better library compatibility.
